Sample TODO App
======

This is a sample project to demonstrate how to write Angular 1 app in the component oriented way.

Code is written in ES2015 and bundled using Webpack.

To get started:

1. checkout code to somewhere
2. install node
3. install dependencies - goto project folder `npm install`

Now you should have everything required.

Develop
-------
`npm start`

This will start a development server at http://localhost:8080

Page will refresh automatically whenever changes to JS, HTML and CSS files are detected.


Run Tests
---------
`npm test`

This will start karma and run all the tests. It also monitors source codes changes and rerun the tests automatically.


Build
-----
To build the assets for production deployment

`npm run build`

It will run all the tests once first and then compile and minify all the sources for production deployment.
